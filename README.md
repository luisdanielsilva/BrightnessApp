# BrightnessApp ☀️

A streamlined macOS menu bar utility for instant brightness management. Part of the **Single Use Apps** suite.

## The Concept

In a world of bloated software, **BrightnessApp** follows the philosophy of precision. It does one thing perfectly: providing instant access to display brightness levels directly from your menu bar with zero performance overhead.

## Key Features

- **Quick Select Levels**: Instant jumps to 25%, 50%, 75%, and 100% brightness.
- **Apple Silicon Optimized**: Native performance for M1, M2, and M3 Macs.
- **Dynamic UI**: A minimalist sunlight icon that feels native to the macOS menu bar.
- **Zero Footprint**: Built with modern SwiftUI for minimal energy and memory impact.
- **State Restoration**: Intelligently restores brightness levels when needed.

## Installation

1. Download the latest `BrightnessApp.zip` from the [Releases](https://github.com/luisdanielsilva/BrightnessApp/releases) page.
2. Unzip and move `BrightnessApp.app` to your `/Applications` folder.
3. Open the app. You will see a small ☀️ icon in your menu bar.

## Architecture

- **Language**: Swift 6.0
- **Framework**: SwiftUI / MenuBarExtra
- **Deployment Target**: macOS 14.0+
- **Hardware Interaction**: IOKit / Display Services
